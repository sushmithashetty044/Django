Django Setup for windows:

1. Download latest python version.
   Link: https://www.python.org/downloads/

2. Virtual Environment Wrapper (Optional)
   If u want seperate space for your project so that it will not effect other project.  
   a. pip install virtualenvwrapper
   b. mkvirtualenv sush(Any name)

3. pip install django

4. create folder Ex: mkdir project

5. cd project 

6. Start project by using command
   django-admin startproject sush(Any Name)
